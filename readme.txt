The SG-60 style Guide Creator for WordPress was developed specifically with the designer in mind. Create a beautiful Style Guide in no time!

== Description ==
  
The SG-60 makes communicating your ideas fast, easy and displayed within your vision board, with professional formatting. The name, “SG-60” was developed because of the speed and simplicity of the plugin. Now anyone from a seasoned professional designer to a developer just trying to better communicate their color scheme and font choices to their client can easily set up and include a new SG-60 Style Guide for each new client in under 60 minutes!
  
[Visit Plugin Site](http://arcctrl.com/plugins/sg-60)


== Installation ==

1. Download zipped archive of plugin
1. Log into your WordPress dashobard and add the new plugin via upload
1. Activate the plugin
1. Use the easy to use instructions
1. use shortcode `[sg-60 id="xx"]` to display the style guide
